# Это шпаргалка для Backend

Запуск сервера в режиме реального времени:

```
uvicorn backMain:app --reload
```

Зависимости:

* FastAPI

  ```
  pip install fastapi
  ```
* Uvicorn

  ```
  pip install uvicorn
  ```
* Открытие Swagger:

```
 http://127.0.0.1:8000/docs
```

* Открытие ReDoc:

```
 http://127.0.0.1:8000/redoc
```
